Brevo (envío por API) — recomendado

Manda el mail por HTTPS en vez de SMTP, así funciona incluso en hostings (como Render free) que bloquean los puertos de SMTP. Necesitás una cuenta en brevo.com y tu API key (Configuración → SMTP y API → API Keys).

El email remitente tiene que estar validado como "Sender" en tu cuenta de Brevo, si no los mails van a rebotar.

Email de premio

Se manda automáticamente cuando alguien gana un premio con "Enviar mail" tildado. Usá {{premio}} donde quieras que aparezca el nombre del premio ganado, y {{fecha}} donde quieras que aparezca la fecha del día en que se envía el mail (por ejemplo, para "Tu premio es válido hasta el {{fecha}}").
